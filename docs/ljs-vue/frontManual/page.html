生成页面 ​

废弃！

原因如下：

1.个人习惯，觉得该功能不能生成完美的前端页面，且存在一些问题。

2.如果提供此功能，需要后端配合研究，并完成vue3的功能开发，成本高。

3.对于一个前端熟练工来说，快速完成一个常规的增删查改页面基本上就是沾沾贴贴，稍微改改的事情。

4.对于其他较为复杂的业务需要重新完成页面开发，基本不太适用基础的自动生成功能。
